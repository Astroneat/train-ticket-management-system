# Installation (VS Code)

- You need three (3) components to run this project:
+ A MySQL Database
+ A MySQL Connector
+ Java Source Code

## MySQL Database

- Install [MySQL Workbench](https://dev.mysql.com/downloads/workbench/)
- Create a new MySQL connection
- Run the commands in *db_creation.sql* (Check source code). This will create the tables and insert some dummy data for you to play with.

## MySQL Connector

- Download a [MySQL Connector](https://dev.mysql.com/downloads/connector/j/)
- Inside VS Code, look for the **JAVA PROJECTS** section inside **Explorer** pane.
- Add the downloaded mysql-connector.jar to **Referenced Libraries**.

## Source Code

- Run:
```
git clone https://github.com/Astroneat/train-ticket-management-system.git
```
- Open the project in a new folder
- Build Java Project
- You should be good to go
